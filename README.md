# Análisis de Salud Mundial

Este proyecto tiene como objetivo analizar un conjunto de datos relacionados con indicadores de salud a nivel mundial, utilizando herramientas de Python y la librería Pandas.


Se desarrollan procesos de:


-Manipulación de datos

-Limpieza de información

-Análisis estadístico

-Clasificación de variables


Objetivos

Aplicar estructuras básicas de Python (listas, diccionarios, funciones)

Realizar análisis exploratorio de datos con Pandas

Comprender la importancia de la limpieza de datos

Generar insights a partir de información real



🛠️ Tecnologías utilizadas

Python 🐍

Pandas

Jupyter Notebook
